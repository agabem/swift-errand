# SwiftErrand Nigeria

**Your stress. Our legs.**

Hyperlocal on-demand errand marketplace connecting busy Nigerians with verified, nearby runners.

- **index.html** - Main interactive product showcase website
- **styles.css** - Responsive styling with dark/light themes
- **script.js** - Scroll-spy navigation, PIN input handler, interactive features

## 📱 Key Features

- **5 km Hyperlocal Coverage** - Quick response times within 5 km radius
- **Real-time Tracking** - Live GPS tracking with instant notifications
- **Transparent Pricing** - 15% platform fee + ₦250 safety surcharge
- **Secure Handoff** - PIN-based verification + photo proof
- **Mobile-First Design** - Optimized for quick mobile bookings
- **Trust Layer** - NIN/BVN verification, escrow protection, rating system

## 💰 Unit Economics

On a ₦5,000 task:
- Runner gets: ₦4,250 (85%)
- Platform takes: ₦750 (15%)
- No hidden fees or middleman markup
